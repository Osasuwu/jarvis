## Раздел 3: Что AI мог бы делать больше

### Q9 [Checkboxes]
**Что AI мог бы делать ПОЛНОСТЬЮ САМ?**
- Форматирование / линтинг кода
- Генерация коммит-сообщений
- Первичный code review
- Генерация тестов
- Генерация документации
- Сортировка задач (лейблы, приоритеты)
- Подготовка отчётов / саммари
- Поиск и сравнение инструментов по критериям
- Конвертация форматов (3D, документы, данные)
- Мониторинг (CI, обновления зависимостей)

### Q10 [Checkboxes]
**Что AI мог бы делать, если ты проверишь результат?**
- Реализация задачи по описанию (PR на ревью)
- Рефакторинг кода
- Написание RFC / design документов
- Исследование технологий с рекомендацией
- Создание презентаций по данным
- Подготовка 3D ассетов по описанию
- Ответы на вопросы по проекту (для новых людей)

### Q11 [Multiple choice]
**AI создаёт PR / результат, ты только проверяешь. Ок?**
- Да, если результат качественный
- Только для простых задач
- Предпочитаю делать сам, AI помогает
- Зависит от задачи
